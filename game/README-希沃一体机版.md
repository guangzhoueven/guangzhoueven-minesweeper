# 希沃一体机扫雷比赛版使用说明

## 游戏特色

专为希沃一体机优化的多人轮流扫雷比赛游戏，支持2-10名玩家同时参与，点到雷的玩家出局，最后剩下的玩家获胜。

## 快速开始

### 1. 选择游戏模式
打开游戏后，点击"多人比赛"按钮进入多人游戏模式。

### 2. 设置玩家
- 输入参与游戏的人数（2-10人）
- 点击"添加玩家"按钮
- 系统会自动生成对应数量的玩家（玩家1、玩家2...）

### 3. 开始游戏
- 点击"开始比赛"按钮
- 系统会自动选择游戏难度（默认中级）
- 显示当前玩家和剩余玩家数量

## 游戏规则

### 基本规则
1. 玩家轮流点击格子
2. 点击到雷的玩家立即出局
3. 未点到雷则轮到下一位玩家
4. 最后剩下的玩家获胜

### 操作方式
- **鼠标操作**：左键点击格子，右键标记雷
- **触摸操作**：点击格子标记雷，长按直接打开

## 界面说明

### 玩家状态显示
- **绿色边框**：当前玩家
- **灰色删除线**：已出局玩家
- **彩色边框**：不同玩家的标识颜色

### 游戏状态
- **当前玩家**：显示正在操作的玩家姓名
- **剩余玩家**：显示还未出局的玩家数量
- **已出局**：显示所有出局的玩家

## 难度选择

### 基础（9×9，10个雷）
适合初学者和快速游戏

### 中级（16×16，40个雷）
标准难度，适合大多数玩家

### 专家（30×16，99个雷）
高难度，适合经验丰富的玩家

### 满屏
根据屏幕大小自动调整，充分利用希沃一体机大屏幕

## 教学建议

### 课堂使用
1. **分组比赛**：将学生分成若干小组，每组选出代表参赛
2. **策略讨论**：在游戏过程中让学生讨论扫雷策略
3. **逻辑训练**：通过游戏培养学生的逻辑推理能力

### 游戏技巧
1. **数字分析**：教会学生根据数字推断周围雷的位置
2. **边缘策略**：从边缘开始，逐步向中心推进
3. **概率判断**：在不确定时选择概率较低的位置

## 技术优化

### 希沃一体机适配
- **大屏幕优化**：界面自动适配不同尺寸的大屏幕
- **触摸优化**：支持触摸操作，按钮和区域足够大
- **视觉反馈**：清晰的玩家状态和游戏提示

### 性能优化
- **流畅动画**：踩雷和切换玩家时的动画效果
- **响应迅速**：点击和触摸的即时反馈
- **稳定运行**：长时间游戏不卡顿

## 故障排除

### 常见问题
1. **无法点击**：检查当前玩家是否已出局
2. **界面错乱**：刷新页面重新开始
3. **触摸不灵**：确保手指完全接触屏幕

### 技术支持
如遇到问题，请检查：
- 浏览器是否支持HTML5 Canvas
- JavaScript是否已启用
- 网络连接是否正常
